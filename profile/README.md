# ÆthronOps

**Production-ready Azure Terraform stacks with verifiable compliance proof.**

No access to your Azure. No credentials. Just code you own.

## What we do

We generate complete Terraform projects for Azure using [Azure Verified Modules (AVM)](https://azure.github.io/Azure-Verified-Modules/) — Microsoft's official, maintained modules. Every resource is security-hardened and mapped to 9 compliance frameworks:

**MCSB** · **NIS2** · **RGPD** · **CIS** · **ISO 27001** · **SOC2** · **DORA** · **ANSSI** · **EU AI Act**
